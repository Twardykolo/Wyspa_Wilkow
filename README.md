# Wyspa_Wilkow
This is a simple simulation game written using Python and PyGame.
It's goal is to simulate behaviour of wolfs and rabbits on an island.

# How it works
In this game we have 3 factors:
 - Board tiles
 - Rabbits
 - Wolfs
